# Вопрос на Яндекс.Кью

## Вопрос

Какие приложения или библиотеки нужны чтобы создать свою игру, приложение, бота. Что для этого нужно чтобы самому сделать?

## Ответ

Слушай, вопрос очень обширный. Давай сначала представим примерно, насколько он обширен.

**Допустим, хотим сделать игру.**

- Мобильную? Надо искать библиотеки под это, например: Cocos2dx
- Для компьютера и для телелефона? Что-то общее, например: Unity 3d или Unreal Engine
- Только для ПК? Попробуй Godot Engine или Construct 2

**Допустим, хотим сделать бота.**

- Делаем бота в мессенджер? Есть способ программировать под Telegram или Slack, на YouTube есть обучающие видео
- Делаем бота куда-то ещё? Тут скорее всего нужно самим писать, но библиотек под конкретные задачи хватит.

**Допустим, хотим приложение.**

- Для macOS? Есть Xcode и Xamarin.
- Для iOS? Тот же Xcode и Xamarin, а также React Native, Cordova, Angular и другие
- Для Android? Android Studio, а ещё React Native, Cordova и прочие
- Для Windows? Visual Studio, .NET Framework
- Для Linux? Mono Develop, GTK, KDE, AvaloniaUI

Как видишь, вариантов **много**. И всё перечисленное лишь малая часть от возможного.

**Так с чего начать? За что взяться?**

С цели. С того, **что** именно хочется сделать.

После, можно уже смотреть, как это сделать и какие библиотеки можно использовать.

И вообще, может твоя задача настолько проста, что уже есть готовые решения,
или же способы, в которых не нужно будет программировать.
